NAU · Cantigas do Mar — site
=============================

CONTEÚDO DESTA PASTA
  index.html        → a página (não mudes o nome: o GitHub Pages procura "index.html")
  audio/            → os mp3, já renomeados e ligados à página

COMO PUBLICAR NO GITHUB PAGES
  1. Cria um repositório novo (ex.: nau-cantigas) na tua conta rui77jazz-cpu.
  2. Carrega o index.html E a pasta audio/ inteira para o repositório
     (mantém a pasta audio dentro da raiz, ao lado do index.html).
  3. Settings → Pages → Branch: main / (root) → Save.
  4. Ao fim de 1–2 minutos o site fica em:
        https://rui77jazz-cpu.github.io/nau-cantigas
     (substitui "nau-cantigas" pelo nome que deres ao repositório)

ADICIONAR / TROCAR UM ÁUDIO MAIS TARDE
  Basta pôr o mp3 na pasta audio/ com o nome exato que está abaixo.
  Não precisas de mexer no index.html.
